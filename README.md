# CSS – Complete Guide (Basics to Advanced)

## Live Demo
https://poorvika5102.github.io/css-complete-guide/

This repository demonstrates a complete, practical understanding of CSS from
fundamentals to advanced concepts, with a strong focus on layouts,
responsiveness, modern CSS features, and real-world styling.

It is designed to be recruiter-friendly, easy to review, and backed by a live
deployed demo using GitHub Pages.

---

## What This Repository Demonstrates

- Strong CSS fundamentals (selectors, properties, box model)
- Display behavior and positioning
- Modern layout systems (Flexbox and Grid)
- Responsive design using media queries
- Advanced CSS concepts (variables, transitions, animations)
- Proper separation of concerns (HTML + external CSS)
- Real-world styling through a mini project
- Deployment using GitHub Pages
